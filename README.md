# OOP_Task2_Checkers
This is the game Checkers which was written on Java

## Features
- Two game variants: Russian and Spanish checkers
- Multilingual interface (RU/EN/FR)
- Game state saving/loading (JSON)
- Move history recording
- Customizable board and rules

## Architecture
Built with Java Swing using clean OOP principles:
- Separate packages for game logic, UI, and data
- Flexible rule system for different checkers variants
- Internationalization support
- JSON serialization for game persistence

## Future Development
Plans include cross-platform support, network multiplayer, AI opponent, and mobile/web versions.
